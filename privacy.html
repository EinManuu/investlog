Datenschutzerklärung

Zuletzt aktualisiert: Juni 2025

InvestLog ist eine private, nicht-kommerzielle Anwendung. Es werden keine Daten verkauft, kein Tracking durchgeführt und keine Werbung geschaltet.

1. Verantwortlicher

Diese Anwendung wird privat betrieben von:

EinManuu
Österreich
Kontakt: über GitHub (github.com/EinManuu)

2. Welche Daten werden verarbeitet

InvestLog verarbeitet ausschließlich Daten, die du selbst eingibst:

  • Google-Kontodaten (Name, E-Mail-Adresse) zur Authentifizierung via Google Sign-In
  • Von dir erfasste Investmenttransaktionen (Wertpapier, Kurs, Stückzahl, Datum, Gebühren)

Es werden keine weiteren personenbezogenen Daten erhoben.

3. Zweck der Datenverarbeitung

Die erhobenen Daten dienen ausschließlich dem Betrieb der App:

  • Google-Anmeldedaten zur eindeutigen Identifikation deines Accounts
  • Transaktionsdaten zur Darstellung deines Portfolios und der Steuerberechnung

4. Datenspeicherung und Verschlüsselung

Alle Transaktionsdaten werden verschlüsselt in Google Firestore gespeichert. Die Verschlüsselung erfolgt mit AES-256-GCM. Schlüsselverwaltung läuft über Google Cloud KMS (Key Ring: investlog-keyring, Region: europe-west3).

Google selbst verarbeitet Daten gemäß der Google-Datenschutzerklärung.

5. Weitergabe an Dritte

Es erfolgt keine Weitergabe deiner Daten an Dritte. Die App nutzt folgende externe Dienste ausschließlich für ihren Betrieb:

  • Google Firebase / Firestore (Datenspeicherung)
  • Google Sign-In (Authentifizierung)
  • Google Cloud KMS (Schlüsselverwaltung)
  • Yahoo Finance API (Kursdaten-Suche, keine persönlichen Daten übermittelt)
  • Financial Modeling Prep API (ETF-Sektordaten, keine persönlichen Daten übermittelt)

6. Dauer der Speicherung

Deine Daten werden so lange gespeichert, wie dein Account aktiv ist. Du kannst jederzeit die Löschung deiner Daten verlangen.

7. Deine Rechte (DSGVO)

Als betroffene Person hast du folgende Rechte:

  • Auskunft über gespeicherte Daten (Art. 15 DSGVO)
  • Berichtigung unrichtiger Daten (Art. 16 DSGVO)
  • Löschung deiner Daten (Art. 17 DSGVO)
  • Einschränkung der Verarbeitung (Art. 18 DSGVO)
  • Datenübertragbarkeit (Art. 20 DSGVO)
  • Widerspruch gegen die Verarbeitung (Art. 21 DSGVO)

Zur Ausübung deiner Rechte wende dich über GitHub an den Betreiber.

8. Beschwerderecht

Du hast das Recht, bei der österreichischen Datenschutzbehörde (DSB) Beschwerde einzulegen: www.dsb.gv.at

9. Cookies und Tracking

Diese App und Website verwenden keine Cookies, kein Analytics und kein Tracking.
